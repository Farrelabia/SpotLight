2 Pendekatan Implementasi

Pilih mana yang lebih cocok buat Final Project lu

A

Minimalis (Recommended)

Settings: Dark Mode toggle di overflow menu → buka dialog/bottom sheet kecil, toggle pake SwitchMaterial, simpen ke SharedPreferences.

Favorite: Tombol jadi icon doang (love icon), toggle filled/unfilled. State gak disimpan — reset pas app restart. Snackbar "Ditambahkan ke favorit" / "Dihapus dari favorit".

Delete: Tombol dihapus dari layout.

Pros

  • Gampang implementasi
  • Gak perlu Room DB
  • Cukup buat nilai

Cons

  • Favorite gak persist
  • Dark mode mungkin basic
B

Lebih Lengkap

Settings: Dark Mode toggle → SettingsActivity/Fragment baru pake PreferenceFragmentCompat. Lebih "proper" Android pattern.

Favorite: Icon love toggle + state disimpan di SharedPreferences (Set). Tetap ada walau app di-restart.

Delete: Tombol dihapus dari layout.

Pros

  • Favorite persist
  • Settings proper (PreferenceScreen)
  • Lebih impressive

Cons

  • Lebih kerja
  • PreferenceFragment setup extra
  • Favorite state perlu di-pass antar fragment
